# Dataset: Text Chunks from Various Veganism and Animal Rights Repositories

## Overview

This dataset contains text chunks extracted from 213 different GitHub repositories related to veganism and animal rights. The data has been collected from repositories that are either licensed under permissive open-source licenses or for which the owners have signed data-sharing agreements with us. 

Due to the large size of the dataset, it is not directly hosted on GitHub. You can download the full dataset from the following link:

[Download the dataset](https://drive.google.com/file/d/1pbAdEdPwOFenR3pjWXdcLZbgJXBJxtqQ/view?usp=sharing)

### Dataset Structure

The dataset is provided as a CSV file with the following columns:

- **Repo Name**: The name of the GitHub repository.
- **Repo URL**: The link to the GitHub repository.
- **File Path**: The relative file path of the original file in the repository.
- **Chunk Number**: The chunk number of the text from the original file (if the file was split into smaller sections).
- **File Content**: The actual text content from the repository file.

### Scraping Script

We have also included a Jupyter notebook script called `github_scraping.ipynb`, which can be used to gather similar datasets. To use this script, replace the GitHub user and access token with your own credentials. Ensure that you have permissions for all your starred projects before running the script, as it scrapes starred repositories.

> **Important:** Make sure you have the appropriate permissions to scrape and use the data from the repositories you star, as the script will collect data from those repositories.
